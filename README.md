# Sokoline 🚀

Sokoline is an e-commerce platform built for student entrepreneurs and student ventures. It provides a marketplace where students can launch businesses, manage inventory, and handle transactions within the campus ecosystem.

## 🏗️ Project Structure
- **`sokoline-backend`**: Django 6 (DRF) REST API handling multi-variant products, automated slugs, and core business logic.
- **`sokoline-web`**: Next.js 16 (App Router) frontend featuring a synchronous cart system and a dedicated vendor dashboard.

## 🛠️ Tech Stack
- **Frontend**: Next.js 16, Tailwind CSS, Framer Motion, Clerk Auth, Vitest.
- **Backend**: Django 6, Django REST Framework, PostgreSQL, DigitalOcean Spaces.

## 🚀 Status
- **Buyer Flow**: Complete. Product discovery, variant selection, cart management, and checkout are fully functional.
- **Seller Flow**: In Progress. Dashboard, inventory management, and shop settings are implemented. Product listing form is the next milestone.
